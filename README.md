CIP Project Group 103

Analyzing Transfer Patterns and Spending Behavior in European Football Leagues (2009–2021)

Overview:
This project explores transfer spending trends and player movement patterns across Europe’s top football leagues (2009–2021).  
It examines how nationality, position, and league characteristics influence transfer activity and performance outcomes.
